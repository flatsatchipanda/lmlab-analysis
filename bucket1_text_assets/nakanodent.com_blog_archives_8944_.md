# 「キーン！」が怖い時代は終わる？

## 最新の虫歯治療は“削らない・痛くない”時代へ

こんにちは。

岡山なかの歯科・矯正歯科クリニック 院長の中野浩輔です。

皆さんは「虫歯治療」と聞くと、どんなイメージを持たれるでしょうか？

「キーン！！」という歯を削る音、

「ガリガリ」と響く振動、

そして「痛い！怖い！」という不安…。

歯医者が苦手な理由の多くは、この虫歯治療のイメージにあると思います。

しかし実は近年、歯科医療は大きく進化しています。

今の歯科医療のテーマは、

**「なるべく削らない」「なるべく痛くない」「なるべく歯を残す」**

という考え方です。

昔のように「虫歯＝すぐ削る」という時代から、

できる限り歯を保存する“低侵襲治療”の時代へと変わってきています。

本日は、そんな「最新の虫歯治療」について分かりやすくお話しします。

# 虫歯には進行段階がある

まず大切なのは、虫歯には進行度があるということです。

## C0（初期虫歯）

歯の表面が白く濁ったり、少し茶色く変色している状態です。

まだ穴は開いておらず、適切な歯磨きやフッ素塗布で改善することもあります。

つまり、この段階なら「削らなくて済む」可能性があります。

## C1（エナメル質の虫歯）

虫歯が歯の表面のエナメル質にとどまっている状態です。

痛みはほとんどありません。

この段階で発見できれば、治療も最小限で済みます。

## C2（象牙質まで進んだ虫歯）

虫歯が内部の象牙質まで進行した状態です。

冷たいものがしみたり、時々痛みを感じることがあります。

この段階になると治療が必要です。

## C3（神経まで達した虫歯）

虫歯が歯の神経まで進行した状態です。

熱いものがしみる、夜眠れないほどズキズキ痛むなど、強い症状が出ます。

従来なら「神経を取る治療」が必要になるケースが多い段階です。

## C4（歯の頭が崩壊した状態）

歯の頭の部分が無くなり、根だけ残った状態です。

逆に痛みは感じなくなることもあります。

しかし多くの場合、抜歯が必要になります。

# 虫歯治療は「早期発見」が重要

虫歯は早く見つければ見つけるほど、

痛みも少なく、歯を削る量も少なく済みます。

逆に放置すると、

・神経を取る

・歯が割れる

・抜歯になる

・インプラントや入れ歯が必要になる

など、大きな治療へ発展してしまいます。

だからこそ、定期検診が非常に重要なのです。

# 最新の虫歯治療①

## ヒールオゾン治療

近年注目されているのが「ヒールオゾン治療」です。

これは、オゾンガスの高い殺菌力を利用して虫歯菌を除菌する治療法です。

オゾンは、塩素の約7倍とも言われる強い殺菌力を持っています。

初期虫歯（C0〜C1）に対して行うことで、虫歯の進行を抑える効果が期待されています。

特徴は、

・削る量が少ない

・痛みが少ない

・歯を保存しやすい

という点です。

「歯をなるべく削りたくない」という方には非常に魅力的な治療法です。

# 最新の虫歯治療②

## カリソルブ（カリエスプロ）

次に紹介するのが、スウェーデンで開発された「カリソルブ（カリエスプロ）」です。

これは薬剤で虫歯部分だけを柔らかく溶かし、専用器具で優しく取り除く治療法です。

従来のようにタービンで大きく削る必要が少なく、

「キーン！」という不快音や振動をかなり軽減できます。

対象は主にC2の虫歯。

特徴は、

・健康な歯を残しやすい

・痛みが少ない

・麻酔を減らせる可能性がある

という点です。

歯科恐怖症の方にも向いている治療法と言えるでしょう。

# 最新の虫歯治療③

## MTAセメント

これは非常に画期的な材料です。

通常、C3まで進行すると神経を取るケースが多くなります。

しかしMTAセメントを使用することで、神経を保存できる可能性があります。

MTAセメントには、

・高い封鎖性

・殺菌性

・歯髄保護作用

があり、神経近くまで進行した虫歯でも、神経を守れる場合があります。

歯の神経を取ると、

・歯がもろくなる

・将来的に割れやすくなる

・歯の寿命が短くなる

と言われています。

だからこそ、神経を残す治療は非常に重要なのです。

# 未来の虫歯治療

## エナメル質再生ジェル

さらに今、世界中で研究が進んでいるのが「エナメル質再生ジェル」です。

これは歯のエナメル質に近い成分を使い、失われた部分を再生させる研究です。

現在研究中のジェルは、

・歯に密着する

・自然なエナメル質形成を促す

・硬さまで回復を目指す

という驚くべき特徴があります。

将来的には、

「虫歯を削る時代」から

「虫歯を再生する時代」

へ変わる可能性もあります。

歯科医療は今、大きな進化の真っ只中にあるのです。

# 虫歯治療で一番大切なこと

どれだけ治療が進歩しても、一番重要なのは

## 「早期発見・早期予防」

です。

小さい虫歯なら、

・痛みも少ない

・削る量も少ない

・神経も残せる

・歯の寿命も延びる

可能性が高くなります。

逆に我慢して放置すると、どんな最新治療でも限界があります。

# 最後に

虫歯治療は、

「怖い・痛い・削る」という時代から、

「できるだけ残す・守る・再生する」時代へ変わっています。

歯は一度失うと、二度と天然歯には戻りません。

だからこそ、

・定期検診

・早期発見

・予防歯科

が何より大切です。

「最近歯医者に行っていない」

「少ししみるけど放置している」

そんな方こそ、一度早めのチェックを受けてみてください。
