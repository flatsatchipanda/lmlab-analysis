# 採用までの流れ

Recruitment flow### 採用までの流れ

当院の求人応募から本採用までの流れをご紹介します。

気になることがあればお気軽にお問い合わせください。

-
01応募
以下の方法でご応募できます。

086-256-4618 お問合せフォーム郵送書類を郵送して応募する場合は「応募の旨を記載した添え状」と「写真付き履歴書」を同封のうえ下記宛てに郵送ください。

岡山県岡山市北区矢坂東町6-1

なかの歯科・矯正歯科クリニック書類を郵送して応募する場合は「応募の旨を記載した添え状」と「写真付き履歴書」を同封のうえ下記宛てに郵送ください。

〒700-0074 岡山県岡山市北区矢坂東町6-1 なかの歯科・矯正歯科クリニック

#### 医院見学

面接の前に医院見学だけでも、ご予約いただけます。また、ご予約の際に伝えていただければ、見学の後すぐに一次面接を受けることができます。


見学当日は、白衣をご用意しています。動きやすい靴でお越しください。 -
02面接日時の決定
希望に沿って面接日時を調整します。ご都合が悪い日は遠慮なくお知らせ下さい。

-
03１次面接
チーフクラスの歯科衛生士が約30分間の個別面接をします。お互いに肩の力を抜いて固くなりすぎずに話しましょう。

-
04２次面接
基本的に、1次面接から1週間～2週間後に実施いたします。

-
05内定
お互いの条件が一致したら採用となります。

-
06入社
なかの歯科クリニックの新しい力になってください。期待しています！

#### 試用期間

入社後の3ヶ月間は試用期間になります。終了後に正式な雇用契約を結びます。


※雇用契約書は同意した上で、サインした書類をお互い一部ずつ保有します。 -
07新人研修
当院の制度については、チーフから事前に説明があります。実際の業務


や出勤・退勤の方法、設備機器の使い方などは、OJTで先輩が指導しま

す。きちんと段階を踏んで優しく教えますので安心してください。 -
08本採用
試用期間が終了したら、なかの歯科クリニックの正式なスタッフです。チーム一丸で頑張りましょう！
