# What's 院長メシ

Dinner with the director### なかの歯科の院長メシとは

当院での勤務をお考えの方を対象とした企画です。院長の中野浩輔を含めた2〜4人で食事を楽しみながら、

仕事や将来のことについて語りましょう。リクルートサイトや面接では分からない院長の人となりを知れるだけでなく、なかの歯科・矯正歯科クリニックへの理解を深めることができます。

場所は岡山駅周辺で、お寿司、焼肉、フレンチ、イタリアンなど、何でもOKです。

もちろん「院長メシ」なので、お食事代はご馳走いたします。

### 参加方法

-
01
院長メシに参加したい方は、電話・LINEで『院長メシ』ご希望の旨お伝えいただくか、メールフォームで『院長メシ』を選択ください。

院長メシは医療法人なかの歯科矯正歯科クリニック（所在地：岡山市北区）で働きたい方が対象です。

-
02
日程、開催時間、参加者、お料理ジャンル、お店などはご連絡いただいた後で相談の上決定していきます。

場所は岡山駅周辺です。ご自宅から岡山駅までの交通費についてはご自身でのお支払いをお願いいたします。
